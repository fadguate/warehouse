Respaldo

Exportar a Excel

El inventario ahora vive en tu base de datos de Supabase y se sincroniza automáticamente entre todos los que usan la app — ya no necesitas exportar ni importar para trabajar. Estos botones son solo para tener un respaldo en Excel de vez en cuando (por ejemplo, para enviarlo por correo o archivarlo).

Migrar un Excel existente (una sola vez)

Si ya tenías un inventario en Excel antes de usar esta app, puedes cargarlo aquí una vez para subir esos movimientos a la base de datos. Úsalo solo la primera vez, para no duplicar registros.

Solo un Super Administrador puede migrar un inventario desde Excel, para evitar registros duplicados por accidente.
